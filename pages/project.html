EduPlanIt

EduPlanIt is a dynamic model designed to gauge students' current interests. Utilizing the capabilities of the Google Palm2 API, our model interprets inputs—whether they're bullet points, sentences, or paragraphs—into a skill set that the LLM can comprehend. For enhanced accuracy, we've aggregated matching qualities from various online sources for 21 distinct job titles and have trained the Palm2 API using Makersuite.

Computer Science Tools: Python (Palm2, Google API, Flask), HTML, CSS, JavaScript, Amazon Web Services (EC2, IAM), Large Language Models

Product Management Tools: Git

Primitive Search Engine

I developed a search engine with a front-end interface using a Flask framework frontend and a Python backend. By leveraging data-based APIs like Pandas, I was able to clean and organize the data into something that could be parsed. I used the Whoosh API to create a parser for a data file of 10 GB and produces the results of the top 5 articles. This behavior is a mimic of Google Scholar Engine.

Computer Science Tools: Python (Whoosh, Flask, Pandas, NumPy)

Data Storage Tools: JSON

Product Management Tools: Git
